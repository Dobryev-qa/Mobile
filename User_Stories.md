# User Stories

##  Core Product

### US-01
As a user, I want a home-based fitness app, so that I can train without going to the gym.

### US-02
As a user, I want to receive a monthly challenge, so that I stay motivated and train regularly.

### US-03
As a user, I want to see a list of predefined challenges, so that I can choose a challenge that fits my level.

---

##  Challenge Creation

### US-04
As a user, I want to create my own fitness challenge, so that I can customize my workouts.

### US-05
As a user, I want a quick challenge creation flow, so that I can create a challenge in a few simple steps.

### US-06
As a user, I want an advanced challenge creation mode, so that I can fully customize exercises and difficulty.

---

## Workout & Tracking

### US-07
As a user, I want a timer for time-based exercises, so that I can track exercise duration.

### US-08
As a user, I want to see a daily motivational quote, so that I stay inspired to continue training.

### US-09
As a user, I want to see my active and completed challenges, so that I can track my progress.
